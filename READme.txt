Парсер для скачивания и предварительного анализа вакансий с сайта HH.ru. 
Запускается как обычный ноутбук(с самой верхней ячейки): Среда выполнения >> Выполнить всё 
В конце ноутбука есть кнопка "Start", с помощью которой можно менять вакансии.
Нажав на "Ссылка на вакансию", откроем страницу HTML с вакансией в новой вкладке браузера.
